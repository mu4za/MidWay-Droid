MidWay-Droid aims to revolutionize public safety using intelligent aerial drones integrated with artificial intelligence. Our core objective is to monitor, identify, and respond to threats in real-time, focusing on safeguarding women, children, and crowds in public areas. The system integrates drone surveillance, object/person detection, and automated alert mechanisms directly to local authorities.

üßπ Project Blueprint

Phase 1: Build a basic flying prototype using affordable hardware (APM 2.8, brushless motors, and LiPo batteries).

Phase 2: Integrate GPS and camera systems to enable geo-mapping and real-time visual monitoring.

Phase 3: Connect Raspberry Pi with onboard flight controller to run object detection AI models (YOLO/OpenCV).

Phase 4: Implement crowd behavior analysis, suspicious movement detection, and live alerting via GSM modules.

Phase 5: Expand with Lidar, heat sensing, and swarm control for broader surveillance.
